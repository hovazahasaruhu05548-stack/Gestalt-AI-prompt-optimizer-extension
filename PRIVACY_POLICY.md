# Privacy Policy for Gestalt - AI Prompt Optimizer

**Last Updated: 2026-03-09**

## 1. Data Collection / 数据收集
Gestalt does not collect, store, or transmit any personal identification data or browsing history to any private servers.
Gestalt 不会收集、存储或向任何私人服务器传输任何个人身份数据或浏览历史记录。

## 2. Data Processing / 数据处理
All prompt optimizations are processed locally within the extension or sent directly to the AI platforms you interact with (e.g., ChatGPT, Claude, Gemini). We do not have access to your conversations.
所有提示词优化均在插件本地处理，或直接发送至您交互的 AI 平台（如 ChatGPT、Claude、Gemini）。我们无法访问您的对话内容。

## 3. Permissions / 权限说明
- **Storage**: Used only to save your local settings. (仅用于保存您的本地设置)
- **SidePanel**: Used to provide the optimization interface. (用于提供优化界面)
- **Scripting**: Used to inject the optimization button into AI websites. (用于向 AI 网页注入优化按钮)

## 4. Contact / 联系方式
If you have any questions, please contact us via GitHub Issues.
如有任何疑问，请通过 GitHub Issues 与我们联系。
